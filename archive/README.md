# /archive — Historical Stillform Documents

Documents in this directory are **NOT current**. They are preserved for historical context — to track how Stillform's thinking evolved over time — but should NOT be read as current direction.

## Why these docs are here

Each doc in this directory was once active in repo root. As Stillform's framing evolved (culminating in STILLFORM_FRAMING_LAW.md on May 12, 2026), these docs accumulated framing-violating content that the May 12 audit caught. Rather than delete them and lose Arlin's thinking trail, they were moved here.

## Rules for using these docs

1. **Never copy framing language from these docs into current work.** The framing in here predates STILLFORM_FRAMING_LAW.md and is contaminated.
2. **Never quote from these docs as if they're current direction.** They aren't.
3. **Read them only when historical context matters** — e.g., to understand why a past decision was made or which thinking produced a current decision.
4. **STILLFORM_FRAMING_LAW.md is the supreme reference** for what Stillform IS, IS NOT, and what it produces. These archived docs do not override it; they cannot.
